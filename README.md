# Spring-Boot Standalone Java Application

A template of a Java stand alone application with maven using spring-boot ready to download and import in your favorite IDE.

``` bash
├───src
│   ├───main
│   │   ├───java
│   │   │   ├───com
│   │   │   │   └───springboot
│   │   │   │       ├───algorithms
│   │   │   │       ├───datastructures
│   │   │   │       ├───standalone
│   │   │   │       └───threading
│   │   │   └───google
│   │   │       └───foobar
│   │   └───resources
│   └───test
│       └───java
│           └───com
│               └───springboot
│                   ├───algorithms
│                   │   └───test
│                   └───datastructures
│                       └───test
└───target
    ├───classes
    │   ├───com
    │   │   └───springboot
    │   │       ├───algorithms
    │   │       ├───datastructures
    │   │       ├───standalone
    │   │       └───threading
    └───test-classes
        └───com
            └───springboot
                ├───algorithms
                │   └───test
                └───datastructures
                    └───test
```
